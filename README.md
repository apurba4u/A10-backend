# Fable (Backend)
An Express.js/Node.js backend for the Fable ebook platform.

**Live URL:** [https://fable-server.vercel.app](https://fable-server.vercel.app) (example)

## Features
- RESTful API for Users, Ebooks, Transactions, and Authentication.
- MongoDB database integration with Mongoose.
- Secure JWT-based authentication and role-based middleware.
- Stripe webhook integration for payments.

## Tech Stack
- Node.js, Express, MongoDB, Mongoose, JSON Web Token, bcryptjs, Stripe Node SDK, Better Auth server API.

## Setup
1. `npm install`
2. Configure `.env` based on `.env.example`
3. `npm run dev`
